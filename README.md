## 📌 프로젝트 개요

SSG 랜더스를 위한 팬 커뮤니티 플랫폼으로, 경기 일정, 날씨, 경기 요약, AI 요약 등 다양한 기능을 제공합니다.
외부 사이트에서 데이터를 크롤링하고, 이를 사용자에게 보기 좋게 제공하며, 일부 정보는 OpenAI API를 통해 AI 한줄평으로 자동 요약됩니다.

## ✅ 담당 기능 정리

### 1️⃣ 경기 일정 크롤링 및 달력 표시
- `statiz.sporki.com`에서 SSG 경기일정 크롤링 (Jsoup)
-  SSG가 포함된 경기만 필터링
- `game_schedule` 테이블에 저장 후 FullCalendar에 표시
- 경기 결과가 존재하면 해당 날짜 클릭 시 상세 요약 페이지 이동

### 2️⃣ 경기장별 날씨 위젯
- 기상청 단기 실황 API(`getUltraSrtNcst`) 연동
- 당일 SSG 경기장 기준 날씨 요약 및 온도 표시
- 마커 UI + 경기장 이름 + 날씨 정보 연동
- `weather` 테이블 저장 구조: `stadium_id`, `game_date`, `weather_summary`, `temperature`, `fetched_at`

### 3️⃣ 네이버 스포츠 경기기록 크롤링 + AI 요약
- SSG 경기 기록을 네이버 스포츠에서 크롤링 (Jsoup)
- `etcRecords` 기준 경기 요약 정리
- OpenAI GPT API 호출로 **AI 한줄평 자동 생성**
- `review` 테이블에 경기 요약, 한줄평 저장

### 4️⃣ 선수별 AI 한줄평 생성 (데이터는 타 팀원이 수집)
- 직접 크롤링은 하지 않았으며, 수집된 선수 스탯 기반 요약 담당
- GPT API를 통해 시즌 기록을 한줄평으로 요약
- 각 선수 페이지에서 AI 요약 결과 노출

# 2차 🧢 HALO 야구 팬 커뮤니티 SNS

2. 주요 기술 스택
Backend: Node.js (Express), MySQL, JWT, WebSocket
Frontend: React.js, Axios, Bootstrap
기타: OpenWeather API, Socket.IO, Kakao OAuth

역할: 1:1 채팅, 맨션 알림 기능, 광고 게시물 관리 페이지, 구장 날씨 정보 위젯 구현

3. 프로젝트 목표 및 구현 내용
목표:
실시간 소통과 야구 관련 콘텐츠 공유가 가능한 커뮤니티 플랫폼 구축.
1:1 채팅, 멘션, 날씨 정보, 광고 시스템 등 팬 중심의 인터랙션 기능 강화.

핵심 기능 구현:

실시간 WebSocket 기반 1:1 채팅 기능 (방 자동 생성 및 채팅 목록 갱신 포함)

게시글/댓글 내 사용자 맨션 시 알림 발생 및 클릭 시 프로필 이동

관리자 전용 광고 관리 페이지 (게시/수정/삭제 기능 포함)

경기장별 실시간 날씨 정보 표시 (OpenWeather API 연동)

4. 주요 기술 및 트러블슈팅
WebSocket 채팅 구조:
→ 유저 선택 시 자동 방 생성 및 이전 메시지 로드
→ 소켓 이벤트 new_chat_room_created 활용하여 채팅 목록 실시간 반영
→ 메시지 읽음 처리 및 채팅방 별 구분 로직 적용

맨션 기능:
→ Redux 상태 관리로 유저 정보 캐싱
→ 맨션된 닉네임 클릭 시 현재 닉네임과 매칭되는 userId 동적 조회
→ 닉네임 변경 시에도 기존 맨션이 작동하도록 백엔드 연동 로직 개선

날씨 API 연동:
→ 구장별 위/경도 기반 API 호출
→ 날씨 캐싱 시스템으로 요청 횟수 절감 및 속도 향상

광고 게시판:
→ 관리자가 업로드한 광고만 사용자 화면에 노출되도록 권한 필터링
→ 게시글 형식 검증 및 클릭 시 외부 링크 이동 처리

5. 협업 경험 및 성장 포인트
역할:

채팅, 맨션, 날씨, 광고 게시판 등 인터랙션 중심 기능 구현

백엔드와 REST/소켓 연동을 통한 데이터 흐름 조율

관리자용 프론트 화면 직접 구현 및 사용자 경험 개선

협업:

기능별 API 명세 공유 및 파트 간 피드백 주고받으며 개선

Git flow 기반 브랜치 전략 및 PR 리뷰 체계 적용

성장 포인트:

단순 채팅이 아닌, 실시간 알림과 메시지 상태까지 고려한 채팅 시스템 설계

광고/날씨 등 외부 API 연동과 사용자 인터랙션을 동시에 고려한 설계 능력 강화


6. 회고 및 개선 방향
성과:

팬 커뮤니티 특성에 맞춘 실시간 채팅/맨션 기능 안정적으로 구현

구장별 날씨 정보, 광고 기능 등 부가 기능으로 플랫폼 완성도 향상

개선점:

현재 채팅 기능은 1:1 채팅에 한정되어 있어, 단체 채팅 기능 도입 및 이미지·영상 등 멀티미디어 전송 기능 추가가 필요

채팅 UI의 직관성 및 사용자 편의성 향상을 위한 UX 개선 여지 존재

향후 계획:

Socket 기반 그룹채팅, 시스템 알림 등 확장형 메시징 기능 개발 도전

API 설계/보안에 대한 역량을 강화하여 실서비스에서도 유연하게 대응 가능한 개발자로 성장할 것
